Différentes langages pour différentes utilisations

Les languages du web...

La plupart des sites internet présents sur le web sont codés avec ce trio de langages:

  • HTML(HyperText Markup Langage) qui permet la mise en page et la forme du site, contenu, text,image
  • CSS (Cascading Style Sheets) permet de styliser la page, changer les couleurs, les polices,etc...
  • JavaScript/TypeScript, qui permettent de créer des animations et des pages dynamiques

...Le Jeux vidéo...

On utilise différents moteurs de jeux comme Unity (C#), ou Unreal Engine(C++) (et qui possède même son propre langage en block appelé Blueprint !)

...Dans les applications

Chaque langage, afin d’effectuer des actions graphiques, ont besoins d’effectuer des appels spéciaux au système graphique des systèmes d'exploitation.

Pour l’exemple des langages JVM, une librairie est contenue dans le JDK (Java Developement Kit), appelée awt, pour "Abstract Window Toolkit", et qui permet d’effectuer des actions de base sur les systèmes graphiques.
Comme ouvrir une fenêtre, y afficher des éléments ou des pixels, gestions d’events de base etc. JavaFX, est une surcouche d'AWT,
qui permet d’effectuer bien plus de choses, comme afficher une vidéo, gestion plus approfondie des évènements etc...

Voici une infime liste des languages peuvant être utilisés pour créer une application :

  • Languages JVM - AWT
  • Python - TKinter
  • C++ - MFC, ATL
