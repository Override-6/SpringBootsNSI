Les outils de build / VCS (Version Control System)

Les Build Tools
Les Build Tools permettent de gérer les dépendances d'un programme et d'effectuer des actions pendant le compile time.
Elles se complètent au VCS car elles permettent de synchroniser les dépendances et le comportement d'un build entre les différentes machines
Les build tools sont propre à un langbguage.

Git, un outil de versioning
Git est un outil trés répandu et presque incontournable lorsqu'il s'agit d'avancer
sur un projet à plusieurs. Il permet une grande sécurité du code, et les utilisateurs
peuvent retrouver n'importe quelle version d'un fichier. ce qui signifie que,
si un dossier ou fichier est supprimé/corrompu dans un commit, il peut être réstauré
à partir d'un commit plus ancien.

Git est un programme, il dois être hébergé sur internet afin d'être utilisé par
différent utilisateurs et donc réellement avoir un intéret. Il existe des hébergeurs git, 100% gratuit, comme Github qui dispose
de licences par défaut (Apache, OpenSource...), et de protection des droits d'auteur.
ou alors Heroku qui permet d'héberger l'application automatiquement dés lors qu'elle est mise à jour (Constant Integration)
